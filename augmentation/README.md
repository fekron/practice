## How to start

1. Installation:

```bash
git clone https://github.com/fekron/practice.git
```

2.  Run the application:

```sh
python augmentation/main.py
```

3. Run unit tests: (WARNING: tests do not work properly, still in progress)
```sh
python augmentation/tests.py
```
